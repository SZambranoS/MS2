MATLAB files relevant to reproduce the simulations of Fig. 6 of the manuscript. 

https://www.biorxiv.org/content/10.1101/2020.06.27.174995v2

Runs in Matlab R2015a. 

Please cite the proper related publication in case you re-utilize the files, or part of them. 

The simple scripts "Plot..." allow to plot integrate and plot NF-kappaB dynamics for a given parameter set and the dynamics 
of a target gene activation for the model with stripping,  the model with no stripping and the model with the gene refractory state.
